# Legacy

Esta pasta guarda codigos antigos usados como referencia durante a integracao.

## CLEAR_MESA

`CLEAR_MESA` era o app separado que desconectava conversas da mesa. Ele foi mantido apenas como referencia, porque a limpeza agora esta integrada ao app principal com:

- consulta em tempo real;
- filtros;
- selecao explicita;
- confirmacao em caixa antes de executar;
- logs de auditoria.

Nao execute o script legado diretamente em producao.
